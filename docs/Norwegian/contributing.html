Bidra

Vi ønsker at du bidrar til Chainsaw Human Typing. Følg disse trinnene for å bidra til prosjektet:

  1. Forke prosjektet: Gå til Github og lag en fork av prosjektet.
  2. Klon prosjektet: Klon prosjektet til din lokale maskin ved å kjøre følgende kommando i terminalen:
    3. git clone https://github.com/BRUKERNAVN/chainsaw-human-typing.git
  3. Opprett en ny gren: Opprett en ny gren for å jobbe på ved å kjøre følgende kommando:
    4. git checkout -b NY-GREN
  4. Gjør endringer: Gjør endringer i koden og legg til nye funksjoner.
  5. Commit endringer: Commit endringene til grenen din ved å kjøre følgende kommando:
    6. git commit -m "BESKRIVELSE AV ENDRINGER"
  6. Push endringer: Push endringene til din fork ved å kjøre følgende kommando:
    7. git push origin NY-GREN
  7. Åpne en pull request: Gå tilbake til prosjektets originale side på Github og åpne en pull request. Gi en detaljert beskrivelse av endringene dine.
  8. Gjennomgang: Ditt pull request vil bli gjennomgått av prosjektets bidragsytere. Hvis alt er i orden, vil endringene dine bli slått sammen med hovedprosjektet.

Hvis du har spørsmål eller trenger hjelp, ikke nøl med å åpne en issue eller kontakte prosjektets bidragsytere.
