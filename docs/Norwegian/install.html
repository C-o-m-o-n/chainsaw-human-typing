Installering

For å installere Chainsaw Human Typing, følg disse trinnene:

  1. Last ned kjørbar fil: Last ned den nyeste kjørbare versjonen fra Releases siden på Github.
  2. Kjør appen: Når nedlastingen er fullført, pakk ut filen og kjør appen.
  3. Begynn å skrive: Begynn å skrive på tastaturet ditt, og Chainsaw Human Typing vil simulere menneskelig inndata på datamaskinen din.
