Kjør fra kilde

For å kjøre Chainsaw Human Typing fra kildekoden, følg disse trinnene:

  1. Klon depotet: Klon Chainsaw Human Typing depotet fra Github ved å kjøre følgende kommando i terminalen: 
    git clone https://github.com/LyubomirT/chainsaw-human-typing.git
  2. Installer avhengigheter: Naviger til mappen du klonet depotet til og installer avhengighetene ved å kjøre: 
    pip install -r requirements.txt
  3. Kjør appen: Når avhengighetene er installert, kjør appen ved å kjøre: 
    python main.py
  4. Bygg appen: For å bygge appen til en kjørbar fil, kan du bruke PyInstaller. Kjør PowerShell-skriptet som følger med: 
    .\build.ps1
    Dette vil lage en kjørbar fil i mappen dist.
