如何使用 Chainsaw Human Typing

Chainsaw Human Typing 设计用于在一些场景下模拟打字，按照以下步骤以有效使用:

  1. 打开APP: 从开始菜单或桌面运行app, 你会进入app的主页。
  2. 输入文字: 在主页面, 你会看到一个允许你粘贴和输入的文字框。 这段文字会成为被模拟打字输出的文字
  3. 配置设置: 在文字区域下, 你可以配置以下设置:
    • 延迟: 以秒为单位设置开始打字前的延迟
    • 间隔: 设置在每笔之间的间隔。 这决定了整体的打字速度。
    • 每笔字符数: 选择在间隔前写多少个字符
    • 输入 Enter: 如果你打勾了, 应用会在遇到新行时模拟按下Enter键
  4. 开始打字: 点击 "开始打字" 按钮来开始模拟。进度将会显示在下面的进度条里。
  5. 停止打字: 如果你希望中止过程, 点击 "停止打字" 按钮。 这会立马中止打字过程。
