为 Chainsaw Human Typing 做贡献

我们欢迎你为 Chainsaw Human Typing 做贡献! 这是你贡献的方法:

  1. Fork整个仓库: 在你的Github账户上创建一个Fork来做出更改
  2. 克隆你Fork过来的仓库: 把你Fork过来的仓库克隆在你的电脑上: 
    git clone https://github.com/LyubomirT/chainsaw-human-typing.git
  3. 创建一个分支: 为你的改变创建一个分支: 
    git checkout -b my-feature-branch
  4. 做出改变: 实施你的改变和修复，确保你的代码符合现有的编码标准和实践。
  5. 提交你的更改: 带着一个描述提交你的改变: 
    git add .
git commit -m "Add feature X or fix issue Y"
  6. Push 你的改变: 将你的改变push到Fork仓库: 
    git push origin my-feature-branch
  7. 打开一个PR: 去到原仓库并提交一个PR. 为你的改变写一个清晰，具体的描述。
  8. 审阅过程: 你的PR将会被审阅。如果一切都符合要求, 它就会被合并进主仓库。

如果你有任何问题或需要帮助，随时打开一个issue或联系维护者
