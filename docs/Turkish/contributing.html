Chainsaw Human Typing'e Katkıda Bulunma

Chainsaw Human Typing'e katkıda bulunmaktan memnuniyet duyuyoruz! İşte katkıda bulunmanın yolları:

  1. Depoyu Forklayın: Deponun bir kopyasını GitHub'da forklayarak değişikliklerinizi yapın.
  2. Forkladığınız Depoyu Klonlayın: Forkladığınız depoyu yerel makinenize klonlayın: 
    git clone https://github.com/LyubomirT/chainsaw-human-typing.git
  3. Bir Dal Oluşturun: Değişiklikleriniz için yeni bir dal oluşturun: 
    git checkout -b benim-özellik-dalım
  4. Değişiklikleri Yapın: Değişikliklerinizi veya düzeltmelerinizi gerçekleştirin. Kodunuzun mevcut kodlama standartlarına ve uygulamalarına uygun olmasına dikkat edin.
  5. Değişikliklerinizi Taahhüt Edin: Değişikliklerinizi açıklayıcı bir mesaj ile taahhüt edin: 
    git add .
git commit -m "X özelliğini ekle veya Y sorununu düzelt"
  6. Değişiklikleri İtin: Değişikliklerinizi forkladığınız depoya itin: 
    git push origin benim-özellik-dalım
  7. Bir Pull Request Açın: GitHub'daki orijinal depoya gidin ve bir pull request açın. Değişikliklerinizle ilgili net bir açıklama sağlayın.
  8. İnceleme Süreci: Pull request'iniz incelenecektir. Her şey tatmin edici olursa, ana depoya birleştirilecektir.

Herhangi bir sorunuz varsa veya yardıma ihtiyacınız olursa, bir sorun açmaktan veya bakımcılarla iletişime geçmekten çekinmeyin.
