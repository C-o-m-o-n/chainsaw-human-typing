Chainsaw Human Typing'i Kaynaktan Çalıştırma

Chainsaw Human Typing'i kaynak koddan çalıştırmak için aşağıdaki talimatları izleyin:

  1. Python'u Yükleyin: Sisteminize Python 3.6 veya daha yüksek bir sürümün yüklü olduğundan emin olun. Python'u resmi Python web sitesinden indirebilirsiniz.
  2. Depoyu Klonlayın: Git kullanarak depoyu klonlayın. Terminalinizi açın ve şu komutu çalıştırın: 
    git clone https://github.com/LyubomirT/chainsaw-human-typing.git
  3. Bağımlılıkları Yükleyin: Proje dizinine gidin ve gerekli bağımlılıkları yükleyin: 
    pip install -r requirements.txt
  4. Uygulamayı Çalıştırın: Uygulamayı başlatmak için ana script'i çalıştırın: 
    python main.py
  5. Uygulamayı Derleyin: Uygulamayı çalıştırılabilir bir dosya haline getirmek için PyInstaller'ı kullanabilirsiniz. Sağlanan PowerShell script'ini çalıştırın: 
    .\build.ps1
    Bu, dist klasöründe bir çalıştırılabilir dosya oluşturacaktır.
